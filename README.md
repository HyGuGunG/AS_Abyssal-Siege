# Abyssal Siege — 심해 습격

Grand Cross 스핀오프 미니게임 프로토타입 (Phase 2-B).
발키리가 되어 심해에서 밀려드는 괴물들로부터 성 수문을 지켜내는 Wave 방어전.

## 플레이

- **프로토타입**: [index.html](index.html)
- **기획서 v0.8**: [spec.html](spec.html)
- **다운로드**: [abyssal-siege.zip](abyssal-siege.zip)

## 조작

- `W` `A` `S` `D` — 발키리 이동
- 자동 공격 — 가장 가까운 적에게 검 투척
- `Q` — 바람 질주 (순간이동)
- `E` — 성스러운 충격파 (광역)
- 시설 슬롯 클릭 — 방어 시설 건설 (골드 소모)
- `1` `2` `3` — 카드/시설 선택

## 콘텐츠

- Wave 1~5 생존
- 적 3종 (피라냐 고블린 / 어인 기사 / 그림자 가오리)
- 발키리 카드풀 10종 (일반·희귀·영웅)
- 방어 시설 3종 (석궁 포탑 · 마법 발리스타 · 번개 마법진)
