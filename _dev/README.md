# Abyssal Siege — 심해 습격

Grand Cross 스핀오프 미니게임 프로토타입 (v2.0).
발키리가 되어 심해에서 밀려드는 괴물들로부터 성 수문을 지켜내는 Wave 방어전.

## 플레이

- **웹 플레이**: [index.html](index.html)
- **기획서 v2.0**: [spec.html](spec.html)
- **오프라인 다운로드**: [abyssal-siege.zip](abyssal-siege.zip)

## 조작

- `W` `A` `S` `D` — 발키리 이동
- 자동 공격 — 가장 가까운 적에게 검 투척
- `Shift` — 바람 질주 (순간이동, 이동 중 발동)
- `E` — 성스러운 충격파 (광역)
- 시설 슬롯 접근 — 자원 자동 투입으로 방어 시설 건설
- `1` `2` `3` — 카드/시설 선택

## 콘텐츠

- Wave 1~6 생존 + Wave 7 보스전 (총 7 Wave)
- 적 5종 (피라냐 고블린 / 어인 기사 / 그림자 가오리 / 심해 오우거 / 폭발 슬라임)
- 보스: 해양 크라켄 (3페이즈, 5패턴 + 궁극기)
- 발키리 카드풀 11종 (일반 7 / 희귀 3 / 영웅 1)
- 방어 시설 6종 (석궁 포탑 / 마법 발리스타 / 번개 마법진 / 성수 제단 / 수호 결계 / 서리 정령)
- 로그라이트 카드 드래프트 시스템
- 성 레벨 기반 난이도 시스템
