01 — CONCEPT

왜 이 게임인가

킹샷이 반군으로 만든 것을 우리는 심해 생물로 만든다.
검증된 구조 + AS 세계관에 완벽히 녹아드는 설정.

02 — TRIGGER

진입 방식

플레이어가 선택하는 것이 아니라 게임이 먼저 사건을 만든다.
"내가 자리를 비운 사이 왕국이 공격받았다" — 자연스러운 재접속 동기.

TRIGGER FLOW
1
종료 후 4시간 뒤 재출현
클리어든 실패든 게임 종료 시점 기준으로 4시간 후 다음 습격 발생. 수문 레벨이 높을수록 습격 강도 비례 증가.
종료 기준 타이머 4H 후 재출현
2
습격 경보 알림 수신
푸시 알림 발송 + 영지 화면 빨간 경보 UI 표시.
"심해 생물 탐지 — 성 접근 중"
푸시 알림 영지 경보 UI
3
1시간 응전 제한
알림 발생 후 1시간 내 [습격 대응] 버튼을 눌러야 함.

검 투척 메커닉
검은 공격 범위 내 최근접 적에게 자동 발사·꽂힘·자동 회수. 회수 경로 적에게 추가 데미지. 유저는 포지셔닝으로 검 투척 효율을 결정한다.
GCD (글로벌 쿨다운)
성스러운 충격파·바람 질주는 사용한 스킬의 쿨타임을 공유. 성스러운 충격파 사용 시 대시도 동일 쿨타임 적용 — 큰 스킬일수록 더 긴 판단 공백. 스킬 순서가 전략이 된다.
